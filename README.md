## Local Development
1. Clone the repo:
    ```bash
    git clone https://github.com/isobel-p/my-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd my-website
    ```
3. For debugging:
    ```bash
    python3 app.py
    ```
4. To host it yourself use any WSGI server compatible with Flask.
